Maze 95 JS

Maze 95 JS is a port of the classic Windows 95 screensaver "3D Maze" into a playable web game.

Some things to know:

      I have created the maze95 account for storing maze95-js, I'm Agent-11 / Agent X.

      Maze 95 JS could not be possible with my skills if it wasn't for Three JS, a JavaScript library that makes WebGL development much less head-bashing.
      Maze 95 JS also uses Keydrown, an input library I figured out existed through working on sm64js.
      Collision is still a fully work in progress feature, only a part of the maze has collision in it being the beginning part.
